# Run Configurations

This directory contains IntelliJ IDEA run configurations for the project.

## Available Configurations

- **client_run.xml** - Run configuration for the React frontend development server
- **server_run.xml** - Run configuration for the Node.js backend development server

## Usage

These configurations can be imported into IntelliJ IDEA to quickly start the development servers:

1. Open IntelliJ IDEA
2. Go to Run → Edit Configurations
3. The configurations should appear automatically if the project is opened in IntelliJ IDEA

Alternatively, you can manually import them or they will be detected when opening the project.

## Commands

- **Client**: Runs `npm run dev` in the `client/` directory
- **Server**: Runs `npm run dev` in the `server/` directory

These configurations are shared with the team to ensure consistent development setup.
